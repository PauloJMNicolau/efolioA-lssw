Algumas Categorias dos Animes

A maioria das pessoas realizam uma classificação irrealista dos animes, pois os classificam apenas como sendo um desenho animado que é destinado para crianças. No entanto essa classificação está longe de ser a real.

Os animes, embora sendo desenhos animados, nem sempre são destinados a crianças. Por isso mesmo existem várias formas de os classificar, seja através do público-alvo a que se destinam ou através do tema da sua história.

Esta classificação não é única, podendo um anime pertencer a mais do que uma categoria, sejam estas da mesma tabela ou não. Além destas categorias existem ainda as categorias existentes nas animações de outros países (Ação, Aventura, Comédia, Horror, ...)

Anotações HTML

Utilizei para estruturar a página as tag HTML:

  • "header" para o cabeçalho da página;
  • "section" e "article" para criar os elementos onde é colocado o texto da página;
  • "div" para criar os elementos onde são colocadas as imagens e videos, e algumas secções de texto;
  • "footer" para o rodapé da página;
  • "ul" e "li" para criar a lista sem ordenação;
  • "iframe" para incorporar o vídeo do youtube na página;
  • "p" para criar os paragrafos de texto na página;
  • "img" para adicionar imagens na página;
  • "b" para destacar a negrito algumas palavras;
  • "u" para colocar o sublinhado na frase;
  • o menu de navegação é criado com as tags "ul" e "li" para a estrutura e a tag "a" para o endereço;
  • a tag "a" é também utilizada para colocar hiperligações no meio do texto
  • a tag "title" é usada para colocar o titulo na barra do navegador;
  • a tag "meta" é utilizado para definir o encode dos caracteres para UTF-8;
  • a tag "link" é utilizado para incluir a folha de estilos;
  • a tag "hr" para colocar a barra horizontal;
  • "h1" e "h2" para definir titulos das secções da página;
  • "table" para criar a tabela na página de categorias;
  • "tr" para criar as linhas da tabela;
  • "th" para criar o cabeçalho da tabela;
  • "td" para criar as células da tabela;
  • "i" para colocar em itálico os nomes das categorias na tabela;

Anotações CSS

Utilizei para os estilos da página as propriedades CSS:

  • "background-image" para colocar o fundo na página;
  • "background-attachment" com o valor "fixed" para manter o fundo da página fixo;
  • "background-size" com o valor "cover" para colocar o fundo da página a ocupar todo o ecrã visivel;
  • "background-repeat" com o valor "no-repeat" para que o fundo da página não seja repetido;
  • "padding" para definir nos vários elementos o espaçamento entre o conteudo e o limite do elemento;
  • "margin" para definir o espaço entre os elementos;
  • "background-color" para colocar uma cor como fundo do elemento;
  • "background-position" para centrar a imagem de fundo do header;
  • "box-sizing" com o valor "border-box" para definir em vários elementos que o comprimento do elemento também incluir o "padding" e "margin" dos elementos;
  • "border-radius" para definir um arredondamento das bordas do eleemnto "header" e "footer";
  • "width" para definir o comprimento dos elemtos e "min-width" para definir um tamanho minimo ao redimensionar a página;
  • "height" para definir a altura de um elemento (utilizado em videos e imagens);
  • "font-size" para definir o tamanho do tipo de letra;
  • "font-family" para colocar uma fonte de texto;
  • "opacity" para colocar ou não colocar transparencia nos elementos;
  • "text-align" para fazer o alinhamento do texto;
  • "position" para colocar a posição de um elemento relativa ao local "natural" do elemento;
  • "display" para indicar se o elemento devia ser renderizado como um elemento de bloco ou como um elemento de bloco que é colocado na mesma linha de outro elemento do mesmo tipo; [utilizado no alinhamento dos vários elementos da página]
  • "color" para definir a cor do tipo de letra do texto;
  • "text-decoration" para retirar ou colocar sublinhado nos links;
  • "vertical-align" para definir o alinhamneto vertical entre os elementos da mesma linha; [utilizado no alinhamento dos vários elementos da página]
  • "font-variant" para definir no texto da barra de navegação um estilo texto escrito com letras maúsculas com tamanho reduzido;
  • "list-style" para remover da barra de navegação as decorações padrões das listas
  • "font-weight" para colocar com destaque negrito os links da barra de navegação;
  • as hiperligações (link) foram estilizadas para os vários comportamentos possiveis (ativos, visitados, estado normal e estado com ponteiro sobre o link); ["active", "visited", "link", "hover"]
  • "border" para colocar uma borda na tabela e nas células
